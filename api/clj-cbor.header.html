read-size

(read-size input info)

Reads a size value from the initial bytes of the input stream. Returns either a number, the keyword :indefinite, or calls the error handler on reserved info codes.

write

(write out mtype info)

Writes a header byte for the given major-type and additional info numbers. Returns the number of bytes written.

write-major-int

(write-major-int out mtype i)

Writes a header byte for the given major-type, plus extra bytes to encode the given integer value. Always writes the smallest possible representation. Returns the number of bytes written.
